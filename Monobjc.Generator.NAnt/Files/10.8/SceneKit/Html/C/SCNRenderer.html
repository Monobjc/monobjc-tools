Overview

The SCNRenderer class implements SCNSceneRenderer protocol and allows rendering of a scene into an arbitrary low-level OpenGL context.

Class Methods

rendererWithContext:options:

Creates and returns an SCNRenderer instance with the render target specified by the Core OpenGL context.

+ (SCNRenderer *)rendererWithContext:(void *)context options:(NSDictionary *)options;
Parameters
context

A Core OpenGL render context that is used as the render target (usually a CGLContextObj).

options

An optional dictionary for future extensions.

Return Value

A new instance of SCNRenderer that will use context as the render target.
